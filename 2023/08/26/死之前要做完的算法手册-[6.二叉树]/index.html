二叉树的基本理论基础

二叉树的定义代码

平衡二叉树

平衡二叉树是指左右子树的高度差不超过1，就是平衡二叉树

满二叉树

满二叉树是指一个二叉树只有度为0的叶子结点和度为2的非叶子结点，除了叶子结点的每个结点都有左右子树

完全二叉树

完全二叉树是指除了最后一层是满二叉树，并且最后一层必须都先填满左边

二叉搜索树

二叉搜索树是指结点有数值，二叉搜索树是一个有序树

平衡二叉搜索树

平衡二叉搜索树是指结点有数值的平衡二叉树，左右子树高度差不超过1

二叉树的递归遍历

二叉树分为深度优先搜索和广度优先搜索，我们这里先探究的是深度优先遍历中的利用递归遍历

深度优先搜索，当某一个方向遍历到最后一个结点则结束当前方向，进行下一个方向的遍历

广度优先搜索，当遍历完一层之后再遍历下一层，一层一层搜索

二叉树的递归遍历主要分为：前、中、后序遍历

前序遍历的遍历方式为:中左右

中序遍历的遍历方式为:左中右

后序遍历的遍历方式为:左右中

所以我们记前中后序遍历只要知道这是在说明中间结点的位置

例

如何用递归来遍历二叉树，主要分三步

①判断递归需要的参数以及返回值 ：这里参数为存放遍历之后的集合以及当前结点。返回值是Void,因为遍历结果已经存放在参数里面了

②明确递归的结束条件：深度优先遍历，即当前遍历的元素为空则结束这一深度遍历

③判断单层递归要做什么：将存放遍历值，遍历左子树以及遍历右子树

二叉树的非递归遍历(迭代遍历)

上面我们提到了二叉树的深度优先搜索中的递归遍历方法，而递归遍历通常也能用非递归的方法解决

迭代遍历：用一个栈来模拟二叉树的遍历过程

迭代遍历主要的是处理两件事：遍历结点以及处理结点

我们用前序遍历详细说说这个关系

前序遍历

Leetcode.144题(easy)

思路

我们前序遍历也即是中左右的遍历方式，而栈是一个先进后出的一个数据结构

在这里我们遍历结点和处理结点都是同时的，也即当我们遍历到中结点就要将中结点数据添加list中

而左右子结点入栈应该是先右子树入栈再是左子树入栈，因为LIFO的特性，先处理左子树那就得后入栈

实现

①将根节点添加进栈中

②将中结点弹出，如果结点非空则添加到结果集合中，并且将结点的右子树和左子树入栈

（入栈顺序为中，右，左。处理顺序为中，左，右）

③当处理结束将结果集返回

代码

后序遍历

Leetcode.145题(easy)

思路

后序遍历也即左右中的遍历方式 ，我们前序遍历的遍历方式是中左右，然后先入栈右子树再入栈左子树就能达到处理顺序是中左右。而我们将子树的入栈顺序颠倒，也即入栈左子树再入栈右子树就可以得到中右左的结果集，我们最后再将中右左的结果集反转就能得到左右中的遍历结果。

也即 左右中 == 前序遍历 中左右 ->将子树入栈顺序颠倒 中右左 -> 反转结果集 左右中

实现

①将根节点添加进栈中

②将中结点弹出，如果结点非空则添加到结果集合中，并且将结点的左子树和右子树入栈

（入栈顺序为中，左，右。处理顺序为中，右，左）

③反转结果集(中，右，左 反转成 左，右，中)

中序遍历

Leetcode.94题(easy)

思路

这里遍历结点和处理结点不是同时进行的，也即当我们遍历到这个结点要先去处理左节点，所以我们引入了一个用于遍历的指针

通过指针的不断移动确认当前遍历到的结点，如果当前指针不为空则将结点入栈，再去操作左右子树

如果当前指针为空 说明上一个结点没有左子树 则返回到上个结点将其出栈，再去判断右子树是否为空

实现

①定义一个用于遍历的指针cur，代表着此时遍历到的结点

②如果当前结点不为空 则入栈，将指针指向当前结点的左子树

③如果当前结点为空，则说明左子树为空，返回上一个结点将其弹出实现了左中的路线，再将当前结点指向右子树

④如果当前指针为空并且栈为空说明遍历已经结束 实现了左中右的逻辑

统一迭代法

因为前中后序代码的不一致性，所以我们可以引入一个空指针，这样可以使其迭代法的代码可以能像递归法一样只要移动几个代码的位置就能分别实现前中后序

思想：以中序遍历为假设

当当前结点不为空的时候，将其左右子树和当前结点重新入栈 需要实现左中右的逻辑

  • 假如左右子树都不为空 那么实现栈中 右结点 当前结点 null 左结点

  • 假如左子树为空右子树不为空， 那么栈中 右结点 当前结点 null

  • 假如左子树不为空右子树为空， 那么栈中 当前结点 null 左结点

  • 假如左右子树都为空，那么栈中 当前结点 null

当遇见null结点的时候只要pop栈就能找到当前结点，也即维护了一个当前结点后跟着一个空指针标记

这个统一迭代法主要利用了一个空指针来标记当前结点的位置，当结点不为空的时候统一将有效结点入栈，当结点为空时找前一个元素即可找到当前结点。这样代码只需要移动少部分顺序即可完成前中后序的遍历
